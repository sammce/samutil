# samutil
A set of python utility classes to help develop and maintain quality software.

For documentation, refer to the source code.

# Installation
## Using Pip
```bash
  $ pip install samutil
```
## Manual
```bash
  $ git clone https://github.com/sammce/samutil
  $ cd samutil
  $ python setup.py install
```
# Usage
```bash
$ samutil
```
## Test
`test [...files]`
```bash
$ samutil test
```
